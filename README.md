# 🚀Features

*  Barbecue calculation.
*  BMI calculation.
*  Calculating your puppy's age.
*  Calorie calculation.


# 🏃Getting Started

```sh
#  Clone Repository
$ git clone https://github.com/wesleywcr/CalculoDeTudo.git 
```

```sh
# Install Dependencies
$ yarn install
# Run Web aplication
$ yarn start


```
